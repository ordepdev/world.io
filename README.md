### world.io

> Handle streamable events and mark them on a world map in real time.

#### Install

```bash
$ git clone https://github.com/ordepdev/world.io
$ cd world.io
$ npm install
```
#### Usage

```bash
$ npm start
// => Server listening on port 3000
```
